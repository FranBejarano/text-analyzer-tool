# Text Analyzer

Text Analyzer is a web application that provides various statistics and analysis for text input. It calculates the number of characters, words, sentences, paragraphs, pronouns count, average reading time, and longest word in the provided text.

## Installation

Follow these steps to install and run the Text Analyzer application locally:

1. Clone the repository:

   ```

   git clone <https://github.com/FranBejarano/text-analyzer.git>

   ```

2. Navigate to the project directory:

   ```

   cd text-analyzer

   ```

3. Install the dependencies:

   ```

   npm install

   ```

4. Start the development server:

   ```

   npm start

   ```

5. Open your web browser and visit `http://localhost:3000` to access the application.

## Usage

1. Enter or paste your text in the text area provided.

2. As you input or modify the text, the statistics will be automatically updated in the result boxes.

3. The result boxes display the number of characters, words, sentences, paragraphs, pronouns count, average reading time, and longest word.

4. The bottom result box shows the average reading time and the longest word found in the text.

5. Use the social media links in the navbar to visit the developer's LinkedIn profile.
